Hello-ebpf Samples
==================

Samples for using hello-ebpf, see [hello-ebpf](../README.md) for more information.

Requirements
------------

- clang
- libbpf
- bpftool

Or on Ubuntu or Debian:
```sh
    sudo apt install clang libbpf-dev linux-tools-common linux-tools-$(uname -r)
```
